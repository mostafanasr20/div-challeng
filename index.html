Task Manager App

In this full-stack challenge, you will develop a straightforward
task management application, which will involve implementing
essential HTTP methods such as GET, POST, PUT, and DELETE.
This project will provide an opportunity to demonstrate
proficiency in full-stack development by creating a system that
enables users to manage tasks using a range of fundamental
HTTP operations.
